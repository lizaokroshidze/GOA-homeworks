ამისთვის კი ვებ-დეველოპერები იყენებენ სხვადასხვა ინსტუმენტებსა და ტექნოლოგიებს,რომლებიც ხელს უწყობენ ვებ-საიტებსა და აპლიკაციების განვითარებას.მათი გამოყენება დამოკიდებულია იმაზე,თუ რა ტიპის პროექტზე მუშაობენ(Frond-end,back-end ან full-stack).
ეს ტექნოლოგიებია
1.Frond-end - ტექნოლოგიები
•HTML-(HyperText Markup Language): გამოიყენება ვებგვერდის სტრუქტურის შესაქმნელად.
• CSS-(Cascading Style Sheets): გამოიყენება ვებგვერდის ვიზუალური სტილის განსაზღვრისთვის.
• JavaScript-გამოიყენება ვებგვერდების ინტერაქტიულობისთვის (ანიმაციები, რეაქცია მომხმარებლის მოქმედებებზე).
• JavaScript-ჩარჩოები (Frameworks):
• React.js-მომხმარებლის ინტერფეისების შესაქმნელად.
• Vue.js-მარტივი და მოქნილი ჩარჩო.
• Angular-რთული აპლიკაციების შესაქმნელად Google-ის მხარდაჭერით.
• CSS-ჩარჩოები და პრეპროცესორები:
• Bootstrap-მზა დიზაინები და კომპონენტები სწრაფი ვებსაიტების შესაქმნელად.
• SASS/LESS-უკეთესი სტილის მართვა და კოდის ორგანიზაცია.
2. Backend ტექნოლოგიები:
• სერვერული ენები:
• Node.js-JavaScript-ის გამოყენება სერვერზე.
• Python-Django, Flask ჩარჩოებით.
• PHP-WordPress და სხვა სისტემების მხარდაჭერა.
• Ruby on Rails-Ruby-ზე დაფუძნებული ჩარჩო.
• Java-ძლიერი ენები დიდი აპლიკაციების შესაქმნელად.
• მონაცემთა ბაზები:
• SQL-მონაცემთა ბაზები: MySQL, PostgreSQL.
• NoSQL-მონაცემთა ბაზები: MongoDB, CouchDB.
• API-ები და სერვერები: RESTful API, GraphQL, AWS (Amazon Web Services), Firebase.
3.Full-Stack ტექნოლოგიები:
• იყენებენ ორივე (Frontend და Backend) ტექნოლოგიებს, რაც მათ საშუალებას აძლევს მართონ პროექტის ყველა ფაზა. ზოგჯერ იყენებენ ისეთ ჩარჩოებს, როგორიცაა:
• MEAN Stack (MongoDB, Express.js, Angular, Node.js).
• MERN Stack (MongoDB, Express.js, React, Node.js).
